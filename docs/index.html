PySPT

Python Signal Processing Toolbox

面向教学与科研的 Python 信号处理工具箱，参考 MATLAB Signal Processing Toolbox 体系架构设计，提供与 MATLAB 函数对应的 Python 实现。 A teaching-oriented signal processing library for Python, modeled after MATLAB's Signal Processing Toolbox. Provides Python implementations that correspond to MATLAB functions.

示例代码Example

Python 
import numpy as np
import matplotlib.pyplot as plt
from pyspt.waveforms import square, chirp, gauspuls

# Generate time axis
t = np.linspace(0, 1, 1000)

# Square wave (30% duty cycle)
y_sq = square(2 * np.pi * 5 * t, duty=30)

# Linear chirp (1 Hz -> 50 Hz)
y_chirp = chirp(t, f0=1, t1=1, f1=50)

# Gaussian pulse (fc=100 Hz)
t2 = np.linspace(-0.5, 0.5, 1000)
y_gp = gauspuls(t2, fc=100, bw=0.5)

# Plot
fig, axes = plt.subplots(3, 1, figsize=(10, 6))
axes[0].plot(t, y_sq)
axes[1].plot(t, y_chirp)
axes[2].plot(t2, y_gp)
plt.tight_layout()
plt.show()
